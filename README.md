## 🎮 Что это

**RBX Shaders** — премиум-пакет графики для Roblox, который превращает плоскую картинку
в по-настоящему живой мир. Объёмный свет пробивается сквозь листву, вода отражает небо,
а тени становятся мягкими и реалистичными. Игра выглядит как ремастер — а ставится за пару минут.

## 🚀 Как установить

1. Зайди на [**сайт RBX Shaders**](https://rbxshaders.github.io/RBX-Shaders/).
2. Введи свой **код** и **ник в Roblox**, нажми **Отправить**.
3. Посмотри короткий [**видео-гайд**](https://rbxshaders.github.io/RBX-Shaders/#guide) и повтори шаги.
4. Запускай Roblox и наслаждайся новой графикой 🎉

## ❓ FAQ

<details>
<summary><b>Это безопасно для аккаунта?</b></summary>
<br/>
Да. Шейдеры меняют только визуал и не вмешиваются в игровой процесс.
</details>

<details>
<summary><b>Будут ли лагать слабые ПК?</b></summary>
<br/>
Пакет оптимизирован под слабое железо — картинка остаётся плавной.
</details>

<details>
<summary><b>На каких играх работает?</b></summary>
<br/>
На большинстве популярных игр Roblox — свет, вода и цвета улучшаются глобально.
</details>

<details>
<summary><b>Сложно ли ставить?</b></summary>
<br/>
Нет. Есть пошаговый видео-гайд — справится даже новичок за пару минут.
</details>
